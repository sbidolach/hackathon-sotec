# Micro payments for charity organizations

Simple application for payments in charity organizations. Example page available on the URL https://sotec-sndjqgslvv.now.sh

## Requirements

* NodeJS >= v7.0
* Material Design Lite (https://getmdl.io/)
* React (https://facebook.github.io/react/)
* Redux
* Open Payments Cloud

# How to run

* Rename file `/config/config.js.example` to `/config/config.js`
* Run `yarn build`
* Run `yarn start`
* Open webpage `http://localhost:3001`
